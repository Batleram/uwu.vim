## Installation 
# Vim-Plug
- First install <a href="https://github.com/junegunn/vim-plug">vim-plug</a> . 
- ```Plug 'mangeshrex/uwu.vim' ``` 
- Add ```colorscheme uwu``` to .vimrc or init.vim. 

# PackerNvim 
- `use { 'mangeshrex/uwu.vim' }`
