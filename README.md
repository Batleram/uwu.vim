# Installation 
- Vim-Ply 
- First install ![vim-plug](https://github.com/junegunn/vim-plug). 
- ```Plug 'mangeshrex/uwu.vim' ``` 
- Add ```colorscheme uwu``` to .vimrc or init.vim.
